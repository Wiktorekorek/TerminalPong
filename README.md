Polski:
Zagraj w terminalowego Ponga ASCII
Dostosuj liczbę żyć, tempo gry a także poziom trudności.
Dostępne języki w grze: Polski, Angielski, Hiszpański, Niemiecki, Francuski, Portugalski (Brazylijski).

Angielski:
Play terminal ASCII Pong
Adjust the number of lives, game speed, and difficulty level.
Available languages in the game: Polish, English, Spanish, German, French, Brazilian Portuguese.

Hiszpański:
Juega al Pong ASCII en la terminal
Ajusta el número de vidas, la velocidad del juego y el nivel de dificultad.
Idiomas disponibles en el juego: Polaco, Inglés, Español, Alemán, Francés, Portugués (Brasil).

Niemiecki:
Spiele ASCII-Pong im Terminal
Passe die Anzahl der Leben, die Spielgeschwindigkeit und den Schwierigkeitsgrad an.
Verfügbare Sprachen im Spiel: Polnisch, Englisch, Spanisch, Deutsch, Französisch, Brasilianisches Portugiesisch.

Francuski:
Jouez au Pong ASCII dans le terminal
Ajustez le nombre de vies, la vitesse du jeu et le niveau de difficulté.
Langues disponibles dans le jeu : Polonais, Anglais, Espagnol, Allemand, Français, Portugais brésilien.

Portugalski (Brazylijski):
Jogue Pong ASCII no terminal
Ajuste o número de vidas, a velocidade do jogo e o nível de dificuldade.
Idiomas disponíveis no jogo: Polonês, Inglês, Espanhol, Alemão, Francês, Português (Brasil).
